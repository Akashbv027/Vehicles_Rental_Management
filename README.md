# 🚗 Vehicle Rental Management System (Python)

## 📌 Overview
The **Vehicle Rental Management System** is a Python-based application designed to simplify and automate the process of renting vehicles. It helps manage customers, vehicles, and rental transactions efficiently, reducing manual errors and improving business performance.

---

## 🔧 Features
- **Customer Management**
  - Add, update, and view customer details
  - Track rental history for each customer

- **Vehicle Management**
  - Register new vehicles with details (model, type, availability, rental price)
  - Update vehicle status (available, rented, under maintenance)

- **Rental Transactions**
  - Book vehicles for customers with start/end dates
  - Calculate rental charges automatically based on duration and vehicle type
  - Generate invoices/receipts

- **Search & Reports**
  - Search vehicles by type, availability, or price
  - Generate reports on rentals, revenue, and customer activity

---
